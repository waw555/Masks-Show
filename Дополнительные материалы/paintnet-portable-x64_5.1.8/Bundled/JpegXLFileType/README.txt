pdn-jpegxl

A Paint.NET filetype plugin that loads and saves JPEG XL images using
libjxl.

This plugin is bundled with Paint.NET 5.1.5 and later.

If you need the features from a newer version you can still install the
plugin. The plugin will override the bundled version if it has higher
version number.

Installation

1.  Close Paint.NET.
2.  Place JpegXLFileType.dll, JpegXLFileTypeIO_ARM64.dll and
    JpegXLFileTypeIO_x64.dll in the Paint.NET FileTypes folder which is
    usually located in one the following locations depending on the
    Paint.NET version you have installed.

  Paint.NET Version   FileTypes Folder Location
  ------------------- -----------------------------------------
  Classic             C:\Program Files\Paint.NET\FileTypes
  Microsoft Store     Documents\paint.net App Files\FileTypes

3.  Restart Paint.NET.
